# 贡献者行为准则 (Code of Conduct)

## 我们的承诺

为了营造开放、友好的社区环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、种族、性别、性取向、经验水平、教育程度、社会经济地位、国籍、宗教信仰或技术偏好，每个人都享有**无骚扰**的参与体验。

## 我们的准则

有助于创造积极环境的行为包括：

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 以社区的最大利益为重
- 对其他社区成员保持同理心

不可接受的行为包括：

- 性化语言或图像、以及任何形式的性骚扰
- 挑衅、侮辱/贬低性评论、人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，公布他人的私人信息（如住址、邮箱）
- 其他在职业环境中合理认定为不当的行为

## 我们的责任

维护者有义务澄清可接受行为的标准，并对任何不可接受的行为采取公平、适当的纠正措施。

维护者有权且应当删除、编辑或拒绝与其准则不符的评论、提交、代码、wiki 编辑、issue 和 PR，并说明处理理由。

## 适用范围

本准则适用于所有社区空间，以及当个人代表项目或社区在公共空间活动时。代表项目或社区的例子包括：使用官方项目邮箱、通过官方社交媒体账号发帖、作为任命的代表参加线上或线下活动。

## 执行

可通过 **[私信项目维护者 / 邮箱]** 举报骚扰或其他不可接受的行为。所有投诉都会被及时、公正地审阅与调查。维护者有义务尊重举报者的隐私。

## 执行指引

维护者将依据以下准则判断后果：

1. **纠正 (Correction)**：对轻微违规的私下提醒，说明问题及行为规范。
2. **警告 (Warning)**：对单次或系列违规的正式警告。
3. **临时封禁 (Temporary Ban)**：对严重违规的限时禁止参与社区。
4. **永久封禁 (Permanent Ban)**：对屡次或极端违规（如骚扰、歧视、人身攻击）的永久剔除。
